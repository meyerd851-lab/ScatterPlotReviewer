How to Use Scatter Plot Reviewer

A quick guide to metrics, files, and views.

1. Getting Started

  1. Import Data: Load your hydraulic data files (TSF, CSV, Excel) into the appropriate slots in the sidebar.
  2. Map Columns: If prompted, map your file columns to the standard metrics (Time, Level, Velocity, Flow, Rainfall).
  3. Visualize: The data will automatically plot. Toggle files on/off or change the view mode to explore.

2. Importing Data

The app provides designated slots for comparing different datasets:

  • Raw Data: The original, unedited dataset.
  • Edited Data: The processed or corrected dataset.
  • Confirmation: manual field measurements or validation points.
  • Rainfall: Precipitation data for correlation analysis.
  • Events (.evt): Time ranges to quickly zoom into specific storm events.

3. Metrics & Analytics

Analyze the relationships between key hydraulic parameters:

Level / Depth

Vertical height of the flow in the conduit.

Velocity

Speed of the flow moving through the conduit.

Flow Rate

Volumetric quantity of flow passing a point.

Rainfall

Precipitation intensity or volume.

4. Visualization Modes

Scatter Plot

plots Depth vs. Velocity. Crucial for understanding system hydraulics and data quality.

Time Series

Show time series of data over the monitoring period.

Split View

Displays both views simultaneously. Hovering on one highlights the corresponding data in the other.

5. Tools & Tips

  • Manning's Curve: Generate a theoretical flow curve for circular pipes by entering Diameter, Slope, and N value.
  • Pipe Crown: A reference line showing the full pipe depth will appear on the scatter plot when a diameter is set.
  • Interactive Legends: Click on legend items (e.g., "Raw", "Rainfall") to toggle entire dataset groups.
  • Zooming: Click and drag on any plot to zoom in. Double-click to reset the view.
